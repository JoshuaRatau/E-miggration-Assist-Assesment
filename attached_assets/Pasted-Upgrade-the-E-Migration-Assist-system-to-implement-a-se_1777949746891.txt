- Allow users to upload immigration-related documents
- Store documents securely

------------------------------------

1. CREATE DOCUMENT UPLOAD COMPONENT

Create:
/components/DocumentUploader.tsx

- Drag & drop upload
- File picker fallback

- Show uploaded files list

- PDF
- JPG
- PNG

10MB per file

- Medical Evidence

- Other

Option A (quick):
- local /uploads folder

- leadId
- documentType
- fileUrl
- fileName
- mimeType

- file

- documentType
