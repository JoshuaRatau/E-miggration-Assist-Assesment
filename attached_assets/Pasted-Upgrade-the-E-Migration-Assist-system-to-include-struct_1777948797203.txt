1. STANDARDISE LEAD SCORING SYSTEM

Create scoring tiers:

IF score >= 80:
  leadCategory = "HIGH_PRIORITY"

IF score >= 60 AND score < 80:
  leadCategory = "MEDIUM_PRIORITY"

IF score < 60:
  leadCategory = "LOW_PRIORITY"

Store BOTH:
- leadScore (number)
- leadCategory (string)
