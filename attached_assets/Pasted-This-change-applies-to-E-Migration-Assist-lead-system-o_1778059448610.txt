4. OTP VERIFICATION STEP

Add OTP verification before the user can continue to final submission.

Flow:
- On contact screen, user enters email and/or WhatsApp.
- Add button: “Send verification code”.
- User chooses verification method:
  - Email
  - WhatsApp, only if WhatsApp number is provided
- Server generates a 6-digit OTP.
- Store OTP hash, expiry time, attempts count, and channel.
- OTP expires after 10 minutes.
- Add next screen: “Enter Verification Code”.
- User must enter the code before proceeding.
- Max 5 attempts.
- Do not allow final submission until verified.
- For now, if real email/WhatsApp provider is not fully configured, create a development fallback that logs the OTP server-side and shows a safe dev-only message only in non-production.
- In production, never expose OTP on screen.

8. FINAL RESULT PAGE IMPROVEMENT

Final page must be more intelligent and engaging.

Instead of only saying:
“General Status Check / further review”

Generate a short personalised preliminary comment based on user inputs.

Examples:
- If visa expired: explain that the matter requires review because the declared visa status may affect lawful stay or future application options.
- If overstay: explain that the matter requires review because overstay history may require supporting explanation and document checking.
- If passport validity under 6 months: explain that passport validity may affect eligibility or readiness.
- If outside South Africa and wants to apply for visa: explain that the next review will consider the intended visa route and document readiness.
- If undesirable/prohibited: explain that the matter is more complex and needs careful review of letters, dates, and supporting documents.

Rules:
- Do not say approved, rejected, guaranteed, qualify, or eligible.
- Do not give legal advice.
- Use wording like:
  “Based on the information you provided…”
  “Your matter may require further review because…”
  “The next practical step is to review your supporting information and documents.”

Show:
- Reference number
- Preliminary assessment note
- Why further review is needed
- Next step
- Check Status button
- Return Home button
